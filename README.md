# federal_mbt
Автоматизированный Python-пайплайн для сбора, OCR-обработки и классификации PDF-документов о распределении федеральных бюджетных средств. Пайплайн работает в один запуск, автоматически обрабатывает данные за период пятница → пятница, ведёт логирование и легко адаптируется под другие регионы за счёт настраиваемых правил классификации.
